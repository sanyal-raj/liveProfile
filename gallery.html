Edge AI & Video Analytics

RTSP AI Edge Analytics

An end-to-end edge AI analytics platform running on NVIDIA Jetson Orin Nano that ingests RTSP streams from existing IP cameras and performs real-time inference using optimized TensorRT models. Detected events are stored locally with metadata for fast retrieval via a responsive local web dashboard.

  • Ingestion Engine: GStreamer pipelines for H.264/H.265 RTSP stream decoding.
  • AI Pipeline: NVIDIA TensorRT with YOLOv8/v10 on Jetson Orin Nano GPU for real-time scene understanding.
  • Local Storage: NVMe storage with SQLite/PostgreSQL indexing for fast event search.
  • Local Web UI: Responsive dashboard (Streamlit/React) for live feeds, camera config, and AI metadata visualization.
